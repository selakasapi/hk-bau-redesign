## Design Concept

A modern, bold redesign of the HK Bau construction company website featuring an **Industrial Elegance** aesthetic.

### Design Philosophy
- **Raw meets refined**: Industrial construction aesthetics combined with sophisticated, premium design
- **Bold & confident**: Large typography, dramatic layouts, strong visual hierarchy
- **Trust-building**: Clean structure, professional polish, clear communication
- **Modern interactions**: Smooth animations, parallax effects, micro-interactions

## Key Features

### Visual Design
- Dark theme with yellow (#FBBB21) brand accent
- Split-screen hero section with animated statistics
- Grid-based project showcase (blueprint-inspired)
- Parallax scrolling effects
- Animated counters and scroll reveals

### Typography
- **Display Font**: Space Grotesk (bold, modern, geometric)
- **Body Font**: Inter (clean, professional, highly readable)

### Layout Highlights
1. **Hero Section**: Full-screen split layout with stats sidebar
2. **Services Grid**: Card-based with hover effects
3. **Project Showcase**: Masonry-style grid with overlay details
4. **Why Us**: Split section with feature list
5. **Contact CTA**: Prominent call-to-action with pattern background

### Technical Stack
- Pure HTML5, CSS3, Vanilla JavaScript
- No frameworks (lightweight, fast loading)
- Mobile-first responsive design
- Smooth animations using CSS transitions
- Intersection Observer API for scroll reveals
